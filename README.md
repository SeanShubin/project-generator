# Project Generator

## To run directly from local maven repository
Instead of creating or modifying pom.xml files, use this project to regenerate projects from a specification file.
This project will use project-specification.json from the current directory if no arguments are specified.
To execute this project directly from the local maven repository, run
`java -jar ~/.m2/repository/com/seanshubin/project/generator/project-generator-console/0.1.0/project-generator-console-0.1.0.jar`
